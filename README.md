# Petgram 🐾

React Native mobile app for Petgram — Instagram clone for pets.

## Stack
- React Native + Expo
- TypeScript
- React Navigation
- Zustand (state management)
- Axios
- AsyncStorage

## Setup
```bash
npm install
npx expo start
```

## Features
- Auth (register/login/logout with JWT)
- Feed with infinite scroll
- Upload photos from gallery
- Explore — grid of all posts
- Search users
- Profile with stats
- Likes with optimistic update
- Comments
- Follow/Unfollow
- Notifications
- Delete posts


## Backend
Requires [petgram-api](https://github.com/VILATTI/petgram-api) running on `http://127.0.0.1:3000`
